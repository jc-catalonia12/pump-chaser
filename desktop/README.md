# Desktop (Tauri)

The native desktop window and installer packaging for this project.

**All setup, build, CI, and distribution instructions are in the main README:**

→ **[../README.md](../README.md)** — see [Desktop app & installers](../README.md#desktop-app--installers) and [GitHub Actions](../README.md#github-actions-free-tier)

Quick dev commands (from repo root):

```bash
cargo run                              # API + desktop window
cd desktop && cargo tauri dev          # hot reload
cargo run -p mexc-trading-bot          # API only, no window
```
